# 📝 ToDo List Application

A minimalist and responsive ToDo List application built with **React**, **JavaScript**, **HTML**, and **CSS**, using **Firebase Authentication** and **Firebase Firestore** for user management and data storage.

## 🚀 Features

- ✅ Full **CRUD functionality** for managing tasks
- 🔍 **Sorting** by creation date or alphabetically, with **reversible order**
- 🔐 **Authentication** via **email and password** using Firebase Auth
- 🗃️ **Firestore integration** for persistent task storage upon login
- 🌐 **Guest mode** with temporary data storage (persists per browser tab until reload)
- 🎨 **Theme switching** (light/dark mode)
- 📱 **Responsive** and **minimalist UI** for all screen sizes
- 🔄 **Persistent session** support — stay logged in between visits

## 📂 Technologies Used

- [React](https://reactjs.org/)
- JavaScript
- HTML & CSS
- [Firebase Authentication](https://firebase.google.com/products/auth)
- [Firebase Firestore](https://firebase.google.com/products/firestore)

1. Clone the repository:
   ```bash
   https://github.com/DenisMyakishev/ToDoList.git
3. Install dependencies:
   ```bash
   npm install
5. Create a Firebase project and configure authentication & Firestore.
6. Add your Firebase configuration to the project (src->API->firebase.js).
7. Build the project:
   ```bash

9. Run the app:
